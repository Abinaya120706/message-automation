# 📱 WhatsApp Message Automation using Python & Twilio

## 🚀 Project Overview
This mini project automates sending WhatsApp messages using Python and Twilio API.  
It reads contact details from an Excel file and sends customized messages based on user roles.

## 🛠️ Technologies Used
- Python
- Pandas
- Twilio API
- Excel (openpyxl)

## 📂 How It Works
1. Read data from Excel file
2. Identify designation (staff, friend, student, etc.)
3. Generate personalized message
4. Send message via WhatsApp using Twilio API

## 📊 Sample Input
| designation | number |
|------------|--------|
| staff      | 9876543210 |
| friend     | 9123456780 |

## ▶️ How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Add your Twilio credentials in main.py

3. Run the script:
   python main.py

## ⚠️ Note
- Use Twilio sandbox for WhatsApp testing
- Keep your credentials secure
